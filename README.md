# 🧠 KnowMe — AI 自我认知引擎

> 不是任务管理器，不是 AI 规划师。
> 你记录日常，AI 帮你发现自己都不知道的行为模式，建立一份活的「个人操作说明书」。

---

## ✨ 功能亮点

- 📝 **每日记录** — 周历 + TodoList + 感受日记，AI 实时分析情绪与睡眠
- 🔍 **行为洞察** — AI 从你的数据中发现模式，以提问方式验证假设
- 📖 **个人手册** — 综合已确认模式，生成因果链、循环分析、最佳干预点
- 📜 **我的故事** — AI 撰写传记 + 人物画像（核心信念、人生定理、骄傲时刻）
- ⚙️ **灵活配置** — 支持 DeepSeek / 智谱双 AI 服务商，中英双语切换

## 🏗️ 技术栈

| 层级 | 技术 |
|------|------|
| 前端框架 | Streamlit（多页面架构） |
| AI 引擎 | DeepSeek API / 智谱 GLM-4-Flash |
| 数据存储 | 本地 JSON（自动备份轮转） |
| 设计系统 | Notion 极简风格（Inter 字体） |
| 语言 | Python 3 · 中英双语 |

## 🚀 快速开始

```bash
# 1. 安装依赖
pip install streamlit requests

# 2. 启动应用
streamlit run main.py

# 3. 首次使用
#    进入「设置」页 → 输入 API Key → 加载种子数据体验完整链路
```

## 🎨 设计系统

KnowMe 采用 Notion 极简风格，统一设计语言贯穿全部页面：

- **配色** — 纯白背景 `#FFFFFF`，紫色强调 `#7C5EE0`，完成绿 `#10B981`
- **字体** — Inter（Notion 同款），全局 Google Fonts 加载
- **卡片** — 1px `#EBEBEB` 边框，8px 圆角，hover 变深
- **按钮** — 主操作紫色填充，次要白底边框，hover 0.15s 过渡
- **空状态** — 邀请式大 Emoji 提示，引导用户开始

## 🧪 种子数据

设置页提供「加载种子数据」功能，一键生成 2 周模拟数据（14 天 × 2-4 任务 + 日记 + 3 条已确认洞察 + 1 条待回答洞察），无需真实使用即可体验完整产品链路。

## 📊 数据安全

所有数据存储在本地 `growth_data.json`，自动轮转保留最近 5 个备份。API Key 仅存储在本地，不会上传。
